# Digital Clock ⏰

A simple and responsive digital clock built using HTML, CSS, and JavaScript.  
It displays real-time hours, minutes, and seconds with an option to switch between 12-hour and 24-hour formats.

---

## 🚀 Features
- Live updating digital clock
- 12-hour / 24-hour format toggle
- Clean and modern UI design
- Responsive layout

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
